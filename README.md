# dotsAndBoxes
Dots and Boxes Game http://en.wikipedia.org/wiki/Dots_and_Boxes

How to play:
# `git clone https://github.com/omarduarte/dotsAndBoxes.git dotsAndBoxes_directory`
# `cd dotsAndBoxes_directory`
# `bower install`
# Open `app/index.html` is your browser

How to test:
# `cd dotsAndBoxes_directory`
# `npm install`
# `karma start`
